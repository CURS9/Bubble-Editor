# Bubble Editor

A lightweight, browser-based editor for adding text and drawings to manga/comic panels.

## Features
- Freehand drawing with adjustable brush size and color
- Draggable, resizable text elements
- Custom fonts including comic-style options
- Text outline support
- Multi-select and group drag
- Undo support
- Export as PNG

## Usage
No installation required. Open directly in your browser:

[Live Demo](https://curs9.github.io/Bubble-Editor/)

Or download `BubbleEditor.html` and `wild.ttf` and open locally in your browser.
